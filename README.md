## 🌐 Hospedagem Gratuita (recomendado)
### 1. **GitHub Pages** (mais fácil)
```
1. Crie repo no GitHub: rafaela-portfolio
2. git init && git add . && git commit -m "Initial portfolio"
3. git remote add origin https://github.com/SEUUSER/rafaela-portfolio.git
4. git push -u origin main
5. Settings > Pages > Source: Deploy from branch main > Save
6. Site live em: https://SEUUSER.github.io/rafaela-portfolio
```

### 2. **Netlify** (drag & drop)
1. netlify.com > New site from Git > Connect GitHub
2. Ou arraste pasta para netlify.com/drop
3. Site live em segundos! (domínio custom possível)

### 3. **Vercel**
```
npx create-next-app@latest (mas este é static) ou vercel.com > Import Git
```

## 🔧 Customizações
- **Foto**: Coloque `images/profile.jpg` (350x350px)
- **Projetos**: Edite links em index.html > projetos
- **Contato**: seu email/LinkedIn/GitHub
- **Form**: Integre [EmailJS](https://emailjs.com) ou Formspree
- **Projetos reais**: Adicione imagens em `images/projetos1.jpg` etc.
